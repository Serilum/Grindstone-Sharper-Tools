<h2>Mod Description</h2>

<p style="text-align:center"><a href="https://curseforge.com/members/serilum/projects" target="_blank" rel="nofollow"><img src="https://raw.githubusercontent.com/Serilum/.data-workflow/main/badges/svg/curseforge.svg" width="200"></a> <a href="https://modrinth.com/user/Serilum" target="_blank" rel="nofollow"><img src="https://raw.githubusercontent.com/Serilum/.data-workflow/main/badges/svg/modrinth.svg" width="200"></a> <a href="https://patreon.com/serilum" target="_blank" rel="nofollow"><img src="https://raw.githubusercontent.com/Serilum/.data-workflow/main/badges/svg/patreon.svg" width="200"></a> <a href="https://youtube.com/@serilum" target="_blank" rel="nofollow"><img src="https://raw.githubusercontent.com/Serilum/.data-workflow/main/badges/svg/youtube.svg" width="200"></a></p>
<p><strong><span style="font-size:24px">Requires the library mod&nbsp;<a style="font-size:24px" href="https://curseforge.com/minecraft/mc-mods/collective" target="_blank" rel="nofollow">Collective</a>.</span></strong><br><br><strong>&nbsp;&nbsp;&nbsp;This mod is part of <span style="color:#008000"><a style="color:#008000" href="https://curseforge.com/minecraft/modpacks/the-vanilla-experience" target="_blank" rel="nofollow">The Vanilla Experience</a></span> modpack and <span style="color:#006b3f"><a style="color:#006b3f" href="https://curseforge.com/minecraft/mc-mods/serilums-rpg-bundle" target="_blank" rel="nofollow">Serilum's RPG Bundle</a></span> mod.</strong><br><span style="font-size:18px">Grindstone Sharper Tools is a minimalistic mod which adds an ability for the grindstone to temporarily increase the damage of tools used on it.</span> <br><br>Sharpen tools by&nbsp;<strong>Sneak+Right-Clicking</strong> any tool on the grindstone. This will give the tool <strong>N</strong>&nbsp;sharpened uses (defined in config).<br><br><br><strong><span style="font-size:20px">Configurable:</span> <span style="color:#008000;font-size:14px"><a style="color:#008000" href="https://github.com/Serilum/.information/wiki/how-to-configure-mods" rel="nofollow">(&nbsp;how do I configure?&nbsp;)</a></span><br></strong><strong>usesAfterGrinding</strong> (default = 32, min 1, max 5000):&nbsp;The amount of sharper uses a tool has after using it on the grindstone.<br><strong>sharpenedDamageModifier</strong> (default = 1.5, min 0, max 100.0)::&nbsp;The damage modifier of sharpened tools.<br><br><strong>sendUsesLeftInChat</strong>&nbsp;(default = true):&nbsp;Sends the sharpened tool user a message at 75%, 50%, 25%, 10%.<br><strong>showUsesLeftInItemName</strong>&nbsp;(default = true):&nbsp;Shows the uses left in the name of the item.<br><br><strong>nameUsesPrefix</strong>&nbsp;(default = "(Sharpened uses: " ):&nbsp;The prefix of the sharpened uses left in the tool name.<br><strong>nameUsesSuffix</strong>&nbsp;(default =&nbsp; ")" ):&nbsp;The suffix of the sharpened uses left in the tool name.<br><br><br><strong>Sneak + Right-click a grindstone with a tool in your hand:<br><picture><img src="https://github.com/Serilum/.cdn/raw/main/projects/grindstone-sharper-tools/a.jpg" alt="0" width="1423" height="827"></picture></strong><br><strong><br>This will sharpen the tool, and configurably add it to the item name.<br><picture><img src="https://github.com/Serilum/.cdn/raw/main/projects/grindstone-sharper-tools/b.jpg" width="684" height="281"></picture></strong><br><strong><br>Attacking any entity will increase the damage with the modifier set in the config:<br><picture><img src="https://github.com/Serilum/.cdn/raw/main/projects/grindstone-sharper-tools/c.jpg" width="1142" height="664"></picture><br></strong><strong><br>It is configurably also possible to receive the uses left in chat:<br></strong><picture><img src="https://github.com/Serilum/.cdn/raw/main/projects/grindstone-sharper-tools/d.jpg" width="720" height="137"></picture></p>
